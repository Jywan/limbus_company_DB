### How to start

##### frontend

```shell
# frontend 디렉토리로 이동
cd frontend

# npm 패키지 설치
npm install

# dev 환경 실행
npm run dev
```

##### backend

```shell
# backend 디렉토리로 이동
cd backend

# npm 패키지 설치
npm install

# index.js 실행
npm run start:dev
```
